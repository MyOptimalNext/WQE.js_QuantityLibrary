تعريف المتغيرات (Variables)

ما هو المتغير؟

المتغير هو مساحة في الذاكرة لتخزين قيمة ما يمكن استخدامها لاحقًا.

أنواع المتغيرات في JavaScript:

  • var: طريقة قديمة وغير مفضلة الآن.
  • let: حديثة وتستخدم عند الحاجة لتغيير القيمة.
  • const: حديثة وتستخدم للقيم الثابتة التي لا تتغير.

أمثلة:

let age = 25; // متغير يمكن تغيير قيمته لاحقًا
const pi = 3.14; // ثابت، لا يمكن تغييره

تعريف الثوابت (Constants)

ما هي الثوابت؟

هي متغيرات قيمها لا تتغير بعد تعريفها باستخدام const.

أمثلة:

const country = "Algeria"; // ثابت، لا يمكن تغييره

أنواع البيانات (Data Types)

لماذا نهتم بأنواع البيانات؟

لمعرفة كيفية التعامل مع القيم المخزنة في المتغيرات.

أ) أنواع البيانات الأساسية (Primitive Types)

1. الأرقام (Number):

لتخزين القيم العددية.

let age = 30; // عدد صحيح
let price = 19.99; // عدد عشري

2. النصوص (String):

لتخزين النصوص.

let name = "John";
let greeting = "Hello, World!";

3. القيم المنطقية (Boolean):

لتخزين القيم المنطقية: إما true أو false.

let isStudent = true; // نعم، هو طالب
let isOnline = false; // لا، غير متصل

4. غير معرف (Undefined):

عندما لا يتم إعطاء المتغير أي قيمة.

let x; // غير معرف

5. فارغ (Null):

يعبر عن قيمة فارغة مخصصة.

let y = null;

ب) أنواع البيانات الغير أساسية (Non-Primitive Types)

1. المصفوفات (Arrays):

تستخدم لتخزين قائمة من القيم.

let colors = ["red", "green", "blue"];

2. الكائنات (Objects):

تستخدم لتخزين بيانات معقدة على شكل أزواج مفتاح وقيمة.

let car = { brand: "Toyota", model: "Corolla", year: 2021 };

التحقق من نوع البيانات بدون تعقيد

يمكن أن نشرح مفهوم "نوع البيانات" عبر كتابة أمثلة مباشرة، دون استخدام دوال مثل console.log أو مصطلحات متقدمة.

مثال:

let number = 42; // نوعه رقم
let name = "Alice"; // نوعه نص
let isAvailable = true; // نوعه منطقي

نشاط عملي للطلاب

1. اطلب منهم تعريف متغيرات بأنواع مختلفة:

let myAge = 20;      // رقم
let myName = "John"; // نص
let isHappy = true;  // منطقي
